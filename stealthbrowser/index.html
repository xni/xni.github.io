Stealth Web Browser

A browser just for you.

Manifesto

Stealth Web Browser is a browser for an experts. It allows you to control every byte send and every JS api called. Any cookie stored and any local storage entry created. It is ultimately extendible. It can not be detected, so it allows no fingerprinting. It is developed with ultimate power and modern technoligies in mind. Save. Extendible. Free.

Some of the features we are going to implement:

  • Based on servo engine
  • No logs are collected
  • Bookmarks API which allows you to use BitTorrent Sync, VCS or any other backend
  • No restrictions on extension distribution
  • Super safe mode - only in-memory opetations
  • Overridig cache rules for better offline experience
  • Low-memory mode

I was expired by vim. It is so powerful and so extendible that I want to create the same for web browsers.
